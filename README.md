# pastebin
Create new paste dengan rest api pastebin

# Installation

$ pkg install python2

$ pip2 install requests

$ git clone https://github.com/wibucode/pastebin

# Cara menggunakan

- Masukkan nama file yang ingin di pastekan ke pastebin
  
  Contoh: index.html (/path/namafile.ext)

- Masukkan nama (bebas)
